== Description ==

This plugin allows you to generate a gravatar favicon for your blog and admin logo included Apple touch icon. Just activate the plugin, and it will add gravatars to your blog template and admin panel automatically, no setting and nothing. But you must enable your avatar(Discussion Settings). (W6A4N4-L26P6-WSH)

== Installation ==
Support Wordpress and WordpressMU

=Standard MU Plugins installation.=

1. Upload 'gravatar-favicon.php' to the '/wp-content/mu-plugins/' directory

2. It's done!

=Standard Plugins installation.=

1. Upload 'gravatar-favicon.php' to the /wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress

3. No more! Just look your site and admin panel. :)

= How do I disable the favicon? =
The favicon is use gravatar option, so you can disabled the gavatar option(Discussion Settings). Then the favicon will turn off.

= I can't get the favicon display my site? =
Make sure you have enabled the gravatar option in Settings -> Discussion Settings(Show Avatars must checked)
